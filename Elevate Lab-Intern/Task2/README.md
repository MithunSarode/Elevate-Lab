# Student Record Management System

A simple Java application for managing student records.

## Features

- **Student Class**: Contains ID, name, and marks fields
- **ArrayList Storage**: Uses ArrayList to store student objects
- **CRUD Operations**: Add, View, Update, and Delete students

## Files

- `Student.java` - Student class with id, name, and marks fields
- `SRMS.java` - Main application with menu-driven interface

## How to Run

1. Compile the Java files:
   ```
   javac Student.java SRMS.java
   ```

2. Run the application:
   ```
   java SRMS
   ```

## Usage

1. Select option 1 to Add a student (enter ID, name, marks)
2. Select option 2 to View all students
3. Select option 3 to Update a student (enter ID, then new name and marks)
4. Select option 4 to Delete a student (enter ID)
5. Select option 5 to Exit

## Requirements

- Java JDK installed
- Command line/terminal access
